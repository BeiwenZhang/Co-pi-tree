# Co-pi-tree Project Page

This is a static project page for:

**Distilling LLM Reasoning into an Interpretable Policy Tree for Human-AI Collaboration**

Open `index.html` directly in a browser, or serve this folder with any static server.

## Add Videos

Put demo videos in `assets/videos/` using these filenames:

- `cramped-room.mp4`
- `coordination-ring.mp4`
- `asymmetric-advantages.mp4`
- `counter-circuit.mp4`
- `forced-coordination.mp4`
- `human-study.mp4`

The page automatically replaces each placeholder with the corresponding video when the file exists.

## Update Links

In `index.html`, update the hero buttons when public links are ready:

- Paper link: currently points to `assets/papers/copitree.pdf`
- Code link: add a button near the existing Paper/Watch demos/BibTeX buttons
- arXiv link: replace or add once the paper is online

Existing paper figures were converted into web-friendly images under `assets/images/`.
