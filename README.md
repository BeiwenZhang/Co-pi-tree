# Co-pi-tree Project Page

This is a static project page for:

**Distilling LLM Reasoning into an Interpretable Policy Tree for Human-AI Collaboration**

This version follows the structure and visual style of the CoFlow project page:

- `index.html`: overview, selected rollout slots, TL;DR, motivation/framework, citation
- `method.html`: method pipeline and contributions
- `results.html`: benchmark, efficiency, refinement, and transfer results
- `video.html`: rollout video gallery
- `conclusion.html`: short conclusion
- `reference.html`: citation and resource links

Open `index.html` directly in a browser, or serve this folder with any static server.

## Add Videos

Put demo videos in `assets/videos/` using these filenames:

- `cramped-room.mp4`
- `coordination-ring.mp4`
- `asymmetric-advantages.mp4`
- `counter-circuit.mp4`
- `forced-coordination.mp4`
- `human-study.mp4`

The page automatically replaces each placeholder with the corresponding video when the file exists. The same files are used by `index.html` and `video.html`.

## Update Links

Update links when public resources are ready:

- Paper link: currently points to `assets/papers/copitree.pdf`
- arXiv link: replace the paper link in `index.html` and `reference.html` once online
- Code link: add a button in the `links` block in `index.html`
